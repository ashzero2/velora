# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), a limited sandbox for trying out app development with Expo

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).

## Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```

This command will move the starter code to the **app-example** directory and create a blank **app** directory where you can start developing.

## Medical & Scientific References

All medical claims, algorithms, and symptom categories in this app are based on peer-reviewed sources and established clinical guidelines. **This app does not provide medical advice.**

### Menstrual Cycle Phases & Timing
- **Cleveland Clinic — Menstrual Cycle** — https://my.clevelandclinic.org/health/articles/10132-menstrual-cycle
  > Normal cycle: 24–38 days, average 28 days. Four phases: menses, follicular, ovulation, luteal. Bleeding 3–7 days typical.
- **Cleveland Clinic — Ovulation** — https://my.clevelandclinic.org/health/articles/23439-ovulation
  > Ovulation ~day 14 of 28-day cycle. Egg survives 12–24 hrs. Sperm survives 3–5 days. BBT rises 0.5–1°F post-ovulation.
- **UCSF Center for Reproductive Health — Normal Menstrual Cycle** — https://crh.ucsf.edu/about-fertility/normal-menstrual-cycle
  > Luteal phase relatively constant ~14 days. Follicular phase variable length.

### Cycle Prediction & Ovulation Estimation
- **ACOG — Menstrual Cycle as Vital Sign (Committee Opinion 651)** — https://www.acog.org/clinical/clinical-guidance/committee-opinion/articles/2015/12/menstruation-in-girls-and-adolescents-using-the-menstrual-cycle-as-a-vital-sign
  > Normal frequency: 24–38 days. Menstrual cycle should be treated as a vital sign.
- **ACOG — Trying to Get Pregnant** — https://www.acog.org/womens-health/experts-and-stories/the-latest/trying-to-get-pregnant-heres-when-to-have-sex
  > Fertile window: ~6 days per cycle. Sperm can live 5 days. Egg survives 12–24 hours.
- **ACOG — Menstrual Cycle Infographic** — https://www.acog.org/womens-health/infographics/the-menstrual-cycle
  > Average cycle 28 days. Ovulation at ~day 14. Progesterone rises post-ovulation.
- **Wikipedia — Calendar-based Contraceptive Methods** — https://en.wikipedia.org/wiki/Calendar-based_contraceptive_methods
  > Standard Days Method (days 8–19 fertile, Georgetown 2002). Knaus-Ogino Rhythm Method. Luteal phase 12–16 days.

### PMS Symptoms & Tracking
- **Cleveland Clinic — PMS (Premenstrual Syndrome)** — https://my.clevelandclinic.org/health/diseases/24288-pms-premenstrual-syndrome
  > Physical: bloating, breast tenderness, headaches, acne, fatigue. Emotional: mood swings, irritability, anxiety, depression. 75% of women experience PMS. Symptoms start 5–11 days before period.
- **Mayo Clinic — PMS Symptoms & Causes** — https://www.mayoclinic.org/diseases-conditions/premenstrual-syndrome/symptoms-causes/syc-20376780
  > Physical: joint/muscle pain, headache, fatigue, bloating, breast tenderness, acne. Emotional: anxiety, mood swings, crying spells, irritability, insomnia, appetite changes, libido changes.
- **IAPMD — Daily Record of Severity of Problems (DRSP)** — https://www.iapmd.org/drsp
  > Validated clinical symptom tracking tool. 21 symptoms rated 1–6 severity. Categories: mood, interest, physical, behavioral. Gold standard for identifying cyclical patterns.

### Premenstrual Disorders
- **ACOG — Management of Premenstrual Disorders (Clinical Practice Guideline, Dec 2023)** — https://www.acog.org/clinical/clinical-guidance/clinical-practice-guideline/articles/2023/12/management-of-premenstrual-disorders
  > Cyclical mood changes and physical symptoms in luteal phase, improving after menstruation. Key symptoms: irritability, anxiety, depressed mood, breast tenderness, bloating, headache.
- **AAFP — Premenstrual Syndrome and PMDD** — https://www.aafp.org/pubs/afp/issues/2025/0400/premenstrual-syndrome-premenstrual-dysphoric-disorder.pdf
  > PMS diagnosed clinically with characteristic luteal phase symptoms resolving during menstruation.

### Cervical Mucus
- **Cleveland Clinic — Cervical Mucus** — https://my.clevelandclinic.org/health/body/21957-cervical-mucus
  > Predictable patterns: dry post-period → sticky → creamy → egg-white (ovulation) → sticky/dry. FAM tracking method with moderate accuracy.

### Cycle Irregularity & Health Patterns
- **FIGO System 1 / Menstrual Disorder Classification** — https://en.wikipedia.org/wiki/Menstrual_disorder
  > Normal cycle: 24–38 days, variation ≤8 days, flow 3–7 days, volume 5–80 mL. Abnormal: <24 or >38 days, variation >8 days, flow >7 days.
- **Cleveland Clinic — Luteal Phase** — https://my.clevelandclinic.org/health/articles/24417-luteal-phase
  > Luteal phase symptoms: mood changes, tender breasts, bloating, acne, appetite changes.

### Wellness & Nutrition by Cycle Phase
- **Cleveland Clinic — Nutrition and Exercise Throughout Menstrual Cycle** — https://health.clevelandclinic.org/nutrition-and-exercise-throughout-your-menstrual-cycle
  > Phase-specific exercise and nutrition recommendations for each cycle phase.
- **Geisinger Health — Menstrual Cycle Syncing** — https://www.geisinger.org/health-and-wellness/wellness-articles/2026/04/02/17/52/menstrual-cycle-syncing
  > Matching diet and exercise to menstrual cycle phases.
- **Harvard Apple Women's Health Study** — https://hsph.harvard.edu/research/apple-womens-health-study/
  > Large-scale study: cramps peak in menstruation, mood changes peak in late luteal, energy peaks at ovulation. Daily tracking improves cycle awareness.
- **MedlinePlus/NIH — PMS Lifestyle Management** — https://medlineplus.gov/ency/article/001505.htm
  > Drink fluids, eat frequent small meals, balanced diet, limit salt/sugar, regular aerobic exercise, calcium/magnesium supplements may help.

### Prediction Accuracy
- **PMC9375297 — Yu et al., 2022** — https://pmc.ncbi.nlm.nih.gov/articles/PMC9375297/
  > Regular menstruators: fertile window prediction accuracy 87.46%, sensitivity 69.30%, specificity 92.00%. Calendar-only methods: low–moderate accuracy.
